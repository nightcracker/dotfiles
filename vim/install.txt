Windows
-------
Find a vimrc that gets run (like %USERPROFILE%\_vimrc) and add something like this:

    source C:\\Program Files\ (x86)\\dotfiles\\vim\\vimrc.vim

Also run vim_caps_remap_install.bat if you want to remap caps lock to escape.
    
Linux
-----
Edit ~/.vimrc to contain:

    source ~/dotfiles/vim/vimrc.vim
