Adding new plugins:
git subtree add --prefix=vim/bundle/<plugin-name> <remote> master --squash

Pulling in updates
git subtree pull --prefix=vim/bundle/<plugin-name> <remote> master --squash

Pulling in all updates on Windows:
for /F "tokens=*" %A in (vim/bundle/remotes.txt) do git subtree pull --prefix=vim/bundle/%A master --squash
