This project uses quite some other projects, as subtrees. An example is vim
plugins. In order to keep those updated we keep all subtrees used in a file
named subtrees.txt, one per line, in the format <prefix> <remote>.

Adding new subtrees:
git subtree add --prefix=<prefix> <remote> master --squash

Pulling in updates
git subtree pull --prefix=<prefix> <remote> master --squash

Pulling in all updates on Windows (run from dotfiles root):
for /F "tokens=*" %A in (subtrees.txt) do git subtree pull --prefix=%A master --squash

Pulling in all updates on Linux (run from dotfiles root):
printf 'git subtree pull --prefix=%s %s master --squash\n' $(<subtrees.txt) | bash
